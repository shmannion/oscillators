# Files 

py\_functions - non-class member functions and corresponding function table

py\_getset - getter/setter methods for private attributes and corresponding table

py\_helpers - functions for converting C++ objects to Python objects (e.g., map to dict) plus other helpers

py\_members - class members (attributes/variables) table. Empty as using get/set approach

py\_methods - class method functions and corresponding table. Functions run as S.function() in Python.

py\_oscillator\_class\_init - class and module initialisation. Constructor/destructor for the Oscillators
                              class and module initialisation such as docstring.
