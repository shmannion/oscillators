# Files

osc\_analysis - functions for output analysis, correlation coefficents etc.

osc\_gettset - for the get/set methods that are exposed to python 

osc\_globals - global variables/settings definitions and getters/setters

osc\_initialisation - system parameter/setting initialisaton

osc\_integration - functions for numerical integration

osc\_kuramoto - functions for kuramoto model simulations

osc\_time\_series - functions for creating time series from model output.
